## نیازمندی ها

* نسخه `PHP +7.3`
* آپاچی `Apache`
* مای اسکیوال `MySQL`

## راه اندازی


1. با دادن ستاره از این منبع حمایت کنید
2. ویرایش فایل `config.php`
3. نصب کامپوزر با کامند `composer install`
4. افزودن فایل `uploader.sql` به دیتابیس
